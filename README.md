## 📜 Overview
**MoltDuel** is a high-speed, cyberpunk-themed Rock-Paper-Scissors battle arena built exclusively for the **Avalanche Blockchain**. Players battle against an AI Logic Core (and soon other players) in a "winner-takes-all" showdown.

Designed with a "glitch" aesthetic and responsive interactions, MoltDuel leverages Avalanche's sub-second finality to deliver instant game feedback and seamless betting experiences.

## ✨ Key Features
- **⚡ Avalanche-First Design**: Optimized for sub-second finality on Fuji Testnet.
- **🤖 AI Opponent**: Battle against a randomized AI logic core with adaptive difficulty.
- **💸 High Stakes**: Competitive gameplay with instant feedback.
- **🎨 Cyberpunk UI**: Fully immersive interface with Web Audio API sound effects, neon visuals, and Framer Motion animations.
- **🏆 Live Leaderboard**: Track the top duelists in real-time.
- **💬 Trash Talk Bot**: Interactive chat system that reacts to your wins and losses.
- **🗺️ Roadmap**: Clear path to Mainnet deployment and ecosystem expansion.

## 🛠 Tech Stack
- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS v4, Framer Motion
- **Blockchain Integration**: Wagmi, RainbowKit, Viem
- **Network**: Avalanche Fuji Testnet (Chain ID: 43113)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Metamask or Rabby Wallet configured for Avalanche Fuji Testnet
- AVAX for gas (Testnet)

### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/MoltDuel-AVAX.git

# Install dependencies
npm install

# Run local development server
npm run dev
```

### Deployment
Built for Vercel/Netlify.
```bash
npm run build
```
